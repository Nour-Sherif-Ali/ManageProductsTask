# ManageProductsTask

Angular app for managing products.

## How to run the project

1. Install dependencies:

   npm install

2. Start the dev server:

   SHORTCUT : 
   ng s -o
   
   npm start

3. Open the app in your browser:

   http://localhost:4200

Optional:
- Run tests: npm test
- Build for production: npm run build

## Test login credentials
